Random Portfolio Dataset and Generator

This page includes a randomly generated portfolio dataset and its generator package. Refer to the links on the left-hand side to download the version you want.

The python script provides a simplified way to generate random variance-covariance matrix for portfolio optimization problems.

The implementation follows the paper of Hirschberger et al. to generate the random variance-covariance matrix (Q). The code also provides return rates (mu) batch sizes (M) and face values (P) in respective folders. It provides a basis that can be modified for any purpose.

Usage

Inside /src folder, call

randomportfolio.py [-h] -n ASSET -f FILENAME [-s SEED] [-e PARAM_E]
                        [-v PARAM_V] [-b PARAM_E_BAR]

Arguments:
  -h, --help            show help message and exit
  -n ASSET, --asset ASSET
                        number of assets
  -f FILENAME, --filename FILENAME
                        file name
  -s SEED, --seed SEED  seed number
  -e PARAM_E, --param_e PARAM_E
                        the e parameter for the method
  -v PARAM_V, --param_v PARAM_V
                        the v parameter for the method
  -b PARAM_E_BAR, --param_e_bar PARAM_E_BAR
                        the e-bar parameter for the method

Example

> python randomportfolio.py -n 200 -f RD1 -s 1122015

Reference

Hirschberger, Markus, Yue Qi, and Ralph E. Steuer. "Randomly generating portfolio-selection covariance matrices with specified distributional characteristics." European Journal of Operational Research 177.3 (2007): 1610-1625.
